My Projects


Billing and Collection System

A Software Engineering project for CSA Residences under Agreville Realty Corporation. The Billing and Collection System aims to digitalize the recording of tenant contracts and occupancy, rent bills and collections, unit maintenance bills and payments, and creation of invoices.

The software was developed using the MERN stack, which includes MongoDB, Express.js, React, and Node.js.

The system is currently undergoing major changes in the process of reviewing and updating its components to ensure it is ready for deployment.

Univents - Web Application

A Multi-Platform Development project using Flutter based on UI designs from Second Years. Univents is a platform which enables Ateneo de Davao University students to explore, view, and join events, both past and upcoming. The app has a mobile and web component. The web component serves as the admin side, where administrators can manage events and organizations through CRUD operations.

The application is currently being developed through Flutter with Supabase as the backend.
